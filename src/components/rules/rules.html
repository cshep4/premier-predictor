Predictions for each match can be added/updated any time before the match kicks off. As the tournament progresses, the knockout matches will be populated with the correct teams, these predictions can also be made at any time up until each match kicks off. Predictions can be changed as often as you wish. Once a match kicks off, the prediction for that game will be locked in and cannot be changed. Scores for knockout matches will be counted after 120 minutes if the game goes to extra time. If the game goes to penalties, the score after 120 minutes will be counted and the result points will be awarded for a draw prediction.
